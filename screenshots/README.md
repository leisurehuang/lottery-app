# 截图说明

本目录应包含以下截图文件，请在README中引用。

## 需要的截图列表

### 1. home-spring.png
- **内容**：春节主题的首页
- **包含**：统计栏（参与人数、奖项等级、已中奖人数、奖品总数）、抽奖进行中区域
- **分辨率**：1920x1080 或 1280x720

### 2. settings-menu.png
- **内容**：侧边设置菜单
- **包含**：导入数据、奖品配置、主题设置、中奖名单、重置数据选项
- **分辨率**：1920x1080 或 1280x720

### 3. prize-config.png
- **内容**：奖品配置页面
- **包含**：添加奖品表单、抽取模式选择、奖品列表
- **分辨率**：1920x1080 或 1280x720

### 4. theme-settings.png
- **内容**：主题设置页面
- **包含**：5个主题卡片、主题说明
- **分辨率**：1920x1080 或 1280x720

### 5. single-mode.png
- **内容**：逐个抽取模式
- **包含**：当前奖项卡片、滚动的人名、开始/停止按钮
- **分辨率**：1920x1080 或 1280x720

### 6. batch-mode.png
- **内容**：一次性抽取模式
- **包含**：多个闪动的人名卡片、停止按钮
- **分辨率**：1920x1080 或 1280x720

### 7. results.png
- **内容**：中奖名单页面
- **包含**：按奖项分组的中奖者列表、导出按钮
- **分辨率**：1920x1080 或 1280x720

### 8. theme-golden.png
- **内容**：金色华章主题效果
- **包含**：金色配色下的抽奖界面
- **分辨率**：1920x1080 或 1280x720

## 截图方法

### macOS
1. 按 `Cmd + Shift + 4` 截取选定区域
2. 按 `Cmd + Shift + 5` 使用截图工具

### Windows
1. 按 `Win + Shift + S` 截取选定区域
2. 使用截图工具或Snipping Tool

### Linux
```bash
# 使用gnome-screenshot
gnome-screenshot -a

# 使用scrot
scrot -s
```

## 注意事项
- 保存为PNG格式
- 确保截图清晰，文字可读
- 建议使用浏览器全屏模式（F11）
- 隐藏浏览器书签栏以获得更好的效果
