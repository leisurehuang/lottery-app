# 🎉 年会抽奖系统

一个基于 React + Vite 的年会抽奖网页应用，支持多等级奖品配置、逐层抽奖、防重复中奖等功能。

## ✨ 功能特性

- 📥 **数据导入** - 支持批量导入员工姓名和工号
- 🎁 **奖品配置** - 灵活配置多等级奖品及数量
- 🎰 **逐层抽奖** - 从低等级到高等级依次抽取
- 🎨 **动画效果** - 滚动抽奖动画，增加仪式感
- 🚫 **防重复中奖** - 已中奖员工自动排除
- 💾 **数据持久化** - LocalStorage 自动保存
- 📊 **结果展示** - 中奖名单统计和可视化
- 📄 **数据导出** - 支持导出 CSV 和 TXT 格式

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

应用将在 `http://localhost:3000` 打开

### 构建生产版本

```bash
npm run build
```

## 📖 使用指南

### 1. 导入员工数据

- 方式一：上传 CSV 或 TXT 文件
- 方式二：直接粘贴文本数据

数据格式：
```
张三,E001
李四,E002
王五,E003
```

### 2. 配置奖品

- 点击"加载预设配置"快速添加常用奖项
- 或手动添加：设置等级、名称和数量
- 等级数字越小越先抽取（如：1=幸运奖，5=特等奖）

### 3. 开始抽奖

- 按照等级从低到高依次抽奖
- 点击"开始滚动"开始，点击"停止"抽取中奖者
- 当前奖项全部抽完后自动进入下一等级

### 4. 查看结果

- 查看按奖项分组的中奖名单
- 查看完整中奖列表
- 导出中奖名单（CSV/TXT）

## 🎯 抽奖规则

1. 按奖项等级从小到大依次抽取
2. 每个奖项抽取配置的数量后自动进入下一等级
3. 已中奖员工不会再次中奖
4. 所有员工都有机会中任意等级奖项

## 📝 数据格式

### 员工数据
```json
{
  "name": "张三",
  "id": "E001"
}
```

### 奖品配置
```json
{
  "level": 1,
  "name": "幸运奖",
  "count": 10
}
```

### 中奖记录
```json
{
  "employeeId": "E001",
  "employeeName": "张三",
  "prizeName": "幸运奖",
  "prizeLevel": 1,
  "timestamp": "2025-02-07T12:00:00.000Z"
}
```

## ⚠️ 注意事项

1. 数据保存在浏览器 LocalStorage 中，清除浏览器缓存会丢失数据
2. 建议在抽奖前导出当前配置作为备份
3. 重置数据操作不可恢复，请谨慎操作
4. 建议使用现代浏览器（Chrome、Firefox、Edge 等）
