📸 年会抽奖系统 - 截图指南

💡 提示：
  • 建议使用浏览器全屏模式（按F11）截图效果更佳
  • 使用Cmd+Shift+4（Mac）或Win+Shift+S（Windows）截取选定区域
  • 保存为PNG格式，分辨率建议1920x1080或1280x720

1️⃣ home-spring.png - 首页（春节主题）

步骤：
  1. 确保选择的是"春节庆典"主题
  2. 首页自动显示统计栏和抽奖区域
  3. 截图包含：导航栏标题、统计栏（4个数据卡片）、抽奖进行中区域
  4. 确保显示红色背景和金色点缀

2️⃣ settings-menu.png - 设置菜单

步骤：
  1. 点击右上角"⚙️ 设置"按钮
  2. 等待侧边菜单滑出
  3. 截图包含：菜单标题、统计数据、所有菜单项（导入数据、奖品配置、主题设置、中奖名单、重置数据）

3️⃣ prize-config.png - 奖品配置

步骤：
  1. 从设置菜单选择"奖品配置"
  2. 截图包含：添加奖品表单、抽取模式选择按钮、奖品列表表格
  3. 确保显示两种抽取模式（逐个抽取/一次性抽取）

4️⃣ theme-settings.png - 主题设置

步骤：
  1. 从设置菜单选择"主题设置"
  2. 截图包含：所有5个主题卡片、主题说明区域
  3. 确保显示不同主题的配色预览

5️⃣ single-mode.png - 逐个抽取模式

步骤：
  1. 在奖品配置中选择"逐个抽取"
  2. 进入首页点击"开始滚动"
  3. 截图包含：当前奖项卡片、大号滚动的人名、停止按钮（绿色）
  4. 确保显示滚动动画效果

6️⃣ batch-mode.png - 一次性抽取模式

步骤：
  1. 在奖品配置中选择"一次性抽取"
  2. 进入首页点击"开始滚动"
  3. 截图包含：多个同时闪动的人名卡片、停止按钮
  4. 确保显示卡片脉动动画效果

7️⃣ results.png - 中奖名单

步骤：
  1. 完成几轮抽奖后，从设置菜单选择"中奖名单"
  2. 截图包含：按奖项分组的中奖者列表、导出按钮
  3. 确保显示中奖者的姓名和工号

8️⃣ theme-golden.png - 金色华章主题

步骤：
  1. 进入主题设置，点击"金色华章"主题
  2. 返回首页查看效果
  3. 截图包含：金色配色的导航栏、统计栏、抽奖区域
  4. 确保显示金色渐变效果

✅ 截图完成后

  1. 将所有PNG文件保存到 screenshots/ 目录
  2. 确保文件名与上述列表一致
  3. README.md会自动引用这些截图
